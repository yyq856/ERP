# ERP系统

## 项目简介

本项目是一个基于Java开发的企业资源规划(ERP)系统后端服务。该系统包含了销售、客户管理、物料管理、订单处理等核心功能模块，使用Spring Boot框架构建，采用MySQL作为数据存储。

主要功能包括：
- 销售订单管理
- 客户与联系人管理
- 物料与库存管理
- 询价单处理
- 业务伙伴关系管理

## 技术栈

- 后端框架：Spring Boot 2.7.x
- 数据库：MySQL 5.7+
- 构建工具：Maven 3.6+
- 编程语言：Java 8+
- ORM框架：MyBatis
- 日志框架：SLF4J + Logback
- API文档：SpringDoc OpenAPI (Swagger UI)
- 其他：Lombok、JWT、Druid连接池
